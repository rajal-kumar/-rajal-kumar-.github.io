What 'scope' is, and how it works in JavaScript?

• When talking about scope in JavaScript, we consider a scope to be a set of variables, objects, and functions that you can access as well. There are two ways to look at scope, they can either be local or global variables/scope.

Local JavaScript Varibles

• When there is a variable decleared within a JavaScript function, this is then addressed as the LOCAL to the function. Then from there Local variable we have a Local scope, the Local scope is only accessible within the function.

• Here is an example of a Local Scope. * Example from w3schools.
// code here can not use carName

function myFunction() {
var carName = "Volvo";

// code here can use carName

}

• The Local vaiables are only active within the function it is presented in. To sum it up a Local variable will be made when the function will be started up, and then delted when the function ia finished.

Global JavaScript Variables

• When there is a variable decleared outside of a function, it is known as GLOBAL. Just like the Local variable a Global variable has a Global Scope, all the scripts and functions will be able to access the global scope as long as its on a web page

• Here is an example of a Global Scope. * Example from w3schools.
var carName = " Volvo";

// code here can use carName
function myFunction() {

// code here can use carName

}
