parentheses ()

• Declare a variable for the function var variable = doSomething()
• You can execute a function with a parameter, this is normally stored in the '()' e.g var variable = doSomething(return)
• Set conditional statements, these are known as loops. There are many types of loops such as for, for/in, while and do/while. E.g for (i = 0; i < 5; i++) {.....
• To a computer, a switch statement and an if/else statement are the same, but a switch statement can be easier for other humans to read. E.g switch (groceryItem) {
• The conditions set will compare to a case. (groceryItem) will compare to case 'apple'.

brackets []

• Usedto create Arrays. myArray = [ 0, 1, 2, 3...]
• Pull out elements from the array, remeber an Array always starts the count from 0. So 1 will be counted from 0 and up. console.log(myArray[0]);

braces {}

• Can be used to make onjects. E.g var testVariable = {"name:" "Rajal"}
• Then create statements. if (condition) {statement};
• Execute the function. function testVariable () {function}
